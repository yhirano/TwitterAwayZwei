TwitterAway::Zwei
---


【主な機能】
TwitterAwayZweiはMicrosoft Windows Mobile 5.0 Pocket PC上で動作するTwitterクラ
イアントです。


【必要環境】
Microsoft Windows Mobile 5.0 Pocket PC
.NET Compact Framework 2.0

.NET Compact Frameworkについては
http://www.microsoft.com/japan/msdn/vstudio/device/netcf/
を参照してください。


【インストール・アップデート】
ZIPファイルを解凍し、TwitterAwayZweiCab.CABをインストールしてください。


【アンインストール】
[スタート]-[設定]-[システム]タブ-[プログラムの削除]から[garage.uraroji.com 
TwitterAway Zwei]を選択して削除してください。


【使い方】
・使い方
メインフォームのテキストボックスに文字を入れ、Updateボタンを押すと自分のステー
タスを更新します。
タイムラインは一定間隔で自動で取得します。

・設定
トップ画面から[Menu]->[Setting]を選択するとTwitterAway::Zwei設定を行うことがで
きます。
[User name]はTwitterのユーザー名を設定します。
[Password]はTwitterのパスワードを設定します。
